FASE COMERCIAL 3

1) Reemplaza la carpeta del proyecto por esta versión.
2) En Supabase ejecuta el archivo: supabase/commercial_leads.sql
3) Luego corre:
   npm install
   npm run dev

Qué agrega:
- Pestaña "PSR para todos"
- Formulario persistente de leads comerciales en Supabase
- Panel admin para revisar, filtrar, actualizar estado y eliminar consultas
